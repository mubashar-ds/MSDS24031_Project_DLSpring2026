## Included Results

The archive contains:

* Training history (CSV)
* Validation metrics
* ROC curves
* Confusion matrices
* Compression robustness curves
* Evaluation metrics
* Figures used in the final report

Organized as:

```text
results/
├── xception_baseline/
├── vit_baseline/
├── dct_xception/
├── cat_xception/
├── dct_cat_xception/
├── consistency_vit/
├── baselines_compression_results/
├── dct_xception_compression_results/
├── cat_xception_compression_results/
└── dct_cat_xception_compression_results/
```

These results reproduce all experiments presented in the project report and provide the data required for further analysis or visualization.

> **Note:** Downloading the results is optional unless you want to reproduce the figures without re-running the experiments.
