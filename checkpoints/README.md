# Model Checkpoints

The trained model checkpoints are not stored in this repository because they exceed GitHub's recommended file size.

## Download

Download all pretrained checkpoints from Google Drive:

* **Google Drive:** 

    * https://drive.google.com/drive/folders/1ZDe9lVPppD20721ZOQDCJpDfziS03dil?usp=sharing

## Available Checkpoints

```text
best_xception65.pth
best_vit_b16.pth
best_dct_xception.pth
best_cat_xception.pth
best_cat_dct_xception.pth
best_vit_consistency.pth
```

These checkpoints correspond to experiments reported in this repository.

## Installation

Extract or copy all checkpoint files into:

```text
checkpoints/
```

## Notes

- Ensure that the checkpoint filenames remain unchanged.
- Place all checkpoint files directly inside the checkpoints/ directory.
- These checkpoints are compatible with the notebooks and evaluation scripts included in this repository.
